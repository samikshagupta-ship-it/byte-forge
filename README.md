# Sentinel AI - Ethical Predictive Safety System

A cutting-edge web application for anonymous incident reporting with AI-powered credibility analysis and predictive safety features.

## Features

- Anonymous incident reporting
- AI credibility scoring
- Interactive crime map with color-coded markers
- Nagpur location focus
- Global and India-specific suicide rate statistics
- Ethical AI predictions
- Dark cybersecurity theme

## Installation

1. Install Python 3.13
2. Clone or download the project
3. Install dependencies: `pip install -r requirements.txt`
4. Run the application: Double-click `run.bat` or run `python main.py`

## Usage

- Access the website at http://localhost:8000
- Submit reports anonymously
- View reports and credibility scores
- Explore the crime map
- Check trends and statistics
